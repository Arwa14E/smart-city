## Project Overview
The **Smart City** project is a web application designed to help citizens explore city services, events, and infrastructure in a centralized and user-friendly platform.  
It was developed as a **Graduation Project with a group of friends**.  
The project demonstrates full-stack development skills, database design, and interactive features.

---

## Features
- Browse city events and locations.  
- Dynamic data display using **AJAX**.  
- Search and filter capabilities.  
- User-friendly interface (UI/UX).  
- Integration with **SQL** database for structured data management.

---

## Technologies Used
- **Frontend:** HTML5, CSS3, JavaScript  
- **Backend:** PHP / Python Flask (depending on implementation)  
- **Database:** MySQL / SQLite  
- **Tools:** Visual Studio Code, Git
